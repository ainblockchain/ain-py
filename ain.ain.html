async getNonce(args: dict) → Any

Gets a current transaction count of account, which is the nonce of the account.

Parameters

args (dict) – May contain a string ‘address’ and a string ‘from’ values. The ‘address’ indicates the address of the account to get the nonce of, and the ‘from’ indicates where to get the nonce from. It could be either the pending transaction pool (“pending”) or the committed blocks (“committed”). The default value is “committed”.

Returns

The nonce of the account.
